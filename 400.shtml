This is an error page.

They describe errors that happen with a site.

The files are .shtml files, but the contents and formatting are usually the same as a regular html file.

The files are named "###.shtml", with the ### being the error code number. They are placed where the index.html is placed.

Most webhosts provide an area where you can easily create these pages. Check the documentation for your webhost.

This error is 400: Bad Request, which happens if your request's syntax is faulty.
